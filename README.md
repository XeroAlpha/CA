# 命令助手
## 简介
一个支持对Minecraft(基岩版)的命令进行补全与辅助输入App。本仓库为App的核心代码。

App工程参见[命令助手Android版](https://gitee.com/projectxero/cadroid)。

### 特性
* 支持命令的智能补全与纠错
* 支持命令在线帮助
* 加入了已汉化的ID表
* 一键粘贴命令
* 支持历史命令与命令收藏夹
* 支持外加命令库来加入支持更多的功能或命令补全
* 支持更换主题样式
